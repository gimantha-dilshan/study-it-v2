**Study-It** is a high-performance, AI-driven educational WhatsApp bot designed to be the ultimate study companion for students. Powered by **Google Gemini 2.5 Flash Lite**, it provides instant homework assistance, conceptual explanations, and step-by-step problem-solving guides—all within the convenience of WhatsApp.

---

## 🚀 Key Features

- **📸 Homework Scanner**: Snap a photo of any math problem, science diagram, or essay prompt. The bot analyzes the image and provides detailed solutions.
- **💬 24/7 AI Tutor**: Ask any question, from historical facts to complex physics theories, and get clear, encouraging, and accurate answers.
- **🧠 Logical Explanations**: It doesn't just give answers; it breaks down the "why" and "how" behind every solution to ensure deep understanding.
- **⚖️ Smart Quota System**:
  - **Free Tier**: 5 AI messages per day.
  - **Registered Tier**: 100 AI messages per day (Connects to the [Registration Portal](https://studyit-register.vercel.app/)).
- **📱 Seamless Connection**: Connect your WhatsApp using a simple **Pairing Code**—no QR scanning required for most setups.
- **🛡️ Admin Dashboard**: Remotely monitor bot performance and broadcast announcements to all users.

## ⚙️ Getting Started

### 1. Prerequisites
- **Node.js** (v18 or higher)
- **Google AI Studio Key**: Get it at [aistudio.google.com](https://aistudio.google.com/)
- **Supabase Account**: Create a project at [supabase.com](https://supabase.com/)

### 2. Environment Configuration
Create a `.env` file in the root directory:

```env
# AI & WhatsApp
GEMINI_API_KEY=your_gemini_key
ADMIN_NUMBER=947xxxxxxxx  # Your phone number with country code

# Supabase Cloud
SUPABASE_URL=your_supabase_project_url
SUPABASE_KEY=your_supabase_anon_key
```

### 3. Web Portal Configuration (Next.js)
Create a `.env.local` file inside the `web/` directory:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
```

### 4. Database Setup
Run the following SQL in your **Supabase SQL Editor** to create the required tables:

```sql
-- 1. Create the Users Table
CREATE TABLE users (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    jid TEXT UNIQUE NOT NULL,
    phone TEXT,
    name TEXT, -- WhatsApp display name
    email TEXT,
    daily_usage INT DEFAULT 0,
    last_usage_date TEXT,
    is_registered BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 2. Create the Messages Table (Chat History)
CREATE TABLE messages (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    jid TEXT REFERENCES users(jid) ON DELETE CASCADE,
    role TEXT NOT NULL, -- 'user' or 'model'
    content TEXT NOT NULL,
    created_at TIMESTAMPTZ DEFAULT NOW()
);
```

### 5. Installation
```bash
# Clone the repository
git clone <your-repo-url>
cd study-it

# Install dependencies
npm install

# Start the bot
npm start
```

## 📲 Connecting to WhatsApp
1. Run `npm start`.
2. Enter your WhatsApp phone number (with country code, e.g., `94771234567`) when prompted.
3. You will receive an 8-character **Pairing Code**.
4. On your phone: **WhatsApp > Settings > Linked Devices > Link a Device > Link with phone number**.
5. Enter the code shown in your terminal.

## 🛡️ Admin Commands
Admins (set via `ADMIN_NUMBER` in `.env`) can use the following commands directly in the WhatsApp chat:

- **`.stats`**: View total users and global message count.
- **`.broadcast [message]`**: Send a notification to all users who have interacted with the bot.
